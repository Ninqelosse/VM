Проект состоит из двух частей. Первая часть - код, который переводит кеплеровы элементы орбиты в декартовы координаты и наоброт - лежит в папке Vychmeh. Код для тестирования лежит в папке Test1. Для перевода из кеплеровых элементов используется функция KeplToDec. Она принимает следующие аргументы: структуру Kepl, состоящую из средней аномалии M, наклонения i, аргумента перицентра w, долготы восходщего узла W, эксцентриситета e и большой полуоси a, и гравитационный параметр mu(все вышеперечисленные переменные имеют тип double). Функция возвращает структуру Dec, поля которой соотвсетвуют x, y, z (радиус-вектор тела), Vx, Vy, Vz (вектор скростей). Для обратного преобразования используется функция DecToKepl, она принимает на вход две структуры типа Vector: радиус-вектор тела и вектор скорости.Также на вход подается гравитационный параметр.Функция возвращает структуру Kepl. Тестирование выполнялось следующим образом: конкретные значения кеплеровых элементов переводятся в декартовы и сравниваются. Для этого в цикле изменяется средняя аномалия (тело как будто движется по эллипсу). Координаты совпадают в пределах погрешностей (для средней аномалии она растет) пока тело не прошло половину эллипса (далее средняя аномалия начинает уменьшаться, а не увеличиваться, как и аргумент перицентра, который есть константа).
